## 接口详情

### 请求地址
/api/user/loginOut

### 请求类型
POST

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
| username | string | 是 | 用户名 | - | - |
| token | string | 是 | 验证 | - | - |

    "code": 0,
