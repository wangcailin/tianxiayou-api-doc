# 获取手机验证码接口接口
>维护人员：王彩霖  
>创建时间：2017-12-02

## 接口简介
获取手机验证码接口接口

### 请求地址
```
/api/user/sendSms
```

### 请求类型
POST

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
| type | number | 是 | 类型 | - |  1找回密码,2注册 |
| mobile | number | 是 | 注册手机号 | - | 15227077282 |

### 正确响应
```javascript
{
    "success": true,
    "code": 0,
    "msg": "验证码发送成功",
    "data": "966472" // 验证码
}
```

### 错误响应
```javascript
{
    "success": false,
    "code": 1,
    "msg": "验证码发送失败",
    "data": null
}
```
