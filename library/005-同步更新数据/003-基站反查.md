# 基站反查接口示例
>维护人员：王彩霖  
>创建时间：2017-12-02

## 接口简介
基站反查接口  

## 接口详情

### 请求地址
/api/customer-flow

### 请求类型
POST

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
| lat | string | 是 | 经度 | - | - |
| lon | string | 是 | 纬度 | - | - |
| n | string | 否 | 返回条数 | - | - |

### 返回正确JSON示例
```javascript
{
    "success": true,
    "code": 0,
    "msg": "ok",
    "data": [
      {
        "mnc": 1,  // 移动网络号
        "lac": 41048,
        "ci": 3477930,
        "location": {
          "lon": 116.341248,
          "lat": 39.535528
        },
        "acc": 1537
      }
    ]
}
```
