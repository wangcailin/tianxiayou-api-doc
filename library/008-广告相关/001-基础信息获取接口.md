>维护人员：王彩霖

## 接口简介
基础信息获取接口  

## 接口详情

### 请求地址
/api/customer-flow

### 请求类型
POST

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
| version | string | 是 | 应用版本 | - | - |

{

  "code": 0,
  "msg": "ok",
  "data": {

            "images": "",  // 信息图片
